# iMessage-IRC
Forward iMessages to IRC (in development)

## Dependencies
 - Xcode 5+
 - NodObjC
 - Mac OS X 10.10+

## Installation
```shell
git clone https://github.com/meyer9/iMessage-IRC
cd iMessage-IRC
npm install
cp settings.example.json settings.json
vim settings.json
```

## Usage
```shell
node irc_forwarder.js
```
